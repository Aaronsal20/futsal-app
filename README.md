# Tournament Management Platform

A full-stack web application to manage sports tournaments including player auctions, team formation, fixtures, live matches, and tournament statistics.

## Features
- Player auctions with live bidding
- Team and budget management
- Match fixtures and live scoring
- Tournament progression and knockout rounds
- Stats: top scorers, champions, awards

## Tech Stack
- Next.js / React
- Supabase (Postgres, Auth)
- Mantine UI

## Why this project?
Built to solve real-world tournament management problems faced during amateur futsal competitions.
